## 전처리 프로세스

### 1️⃣ 결측치 검사
✅ **결과**: 결측치 없음

- 모든 데이터가 완전함
- NaN 처리 필요 없음

### 2️⃣ 이상치 탐지 및 제거 (IQR 방식)

#### Training 데이터
- **이전 행 수**: 105,000
- **제거된 이상치**: 51,638개
- **이후 행 수**: 53,362
- **제거율**: 49.18%

#### Test 데이터
- **이전 행 수**: 45,000
- **제거된 이상치**: 22,195개
- **이후 행 수**: 22,805
- **제거율**: 49.32%

**방법**: IQR (Interquartile Range)
- Q1 (1사분위수)과 Q3 (3사분위수) 계산
- IQR = Q3 - Q1
- 이상치 범위: [Q1 - 1.5×IQR, Q3 + 1.5×IQR]
- 범위 밖의 값 제거

### 4️⃣ 정규화 (Min-Max Normalization)

✅ **방법**: 0-1 범위 정규화
```
정규화된 값 = (값 - 최소값) / (최대값 - 최소값)
```

**결과**:
- 모든 값이 0~1 범위로 변환
- 알고리즘의 수렴 속도 개선
- 특성 간 스케일 차이 제거
