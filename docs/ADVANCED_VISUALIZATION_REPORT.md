### 1️⃣ **Confusion Matrix** - `07_confusion_matrix.png` (107.77 KB)

**무엇을 보여주나?**
- XGBoost 모델의 예측 정확성을 4개 셀로 표현
- 빨강색 그래디언트: 진할수록 값이 많음

**수치 해석**
```
                  예측
실제          No Loan  Loan
---------  ---------  -----
No Loan      2,365    3,482
Loan           358   16,600
```

**주요 인사이트**
- ✅ True Positives: 16,600 (대출자 중 97.89% 맞게 식별)
- ✅ True Negatives: 2,365 (비대출자 중 40.45% 맞게 식별)
- ⚠️ False Positives: 3,482 (비대출자를 대출자로 오분류)
- ✅ False Negatives: 358 (대출자를 놓침 - 아주 적음)

**발표 시 강조**
"우리 모델은 신용 리스크를 거의 놓치지 않습니다. 358명만 놓쳤지만, False Positive는 자유 심사로 조정 가능합니다."

### 2️⃣ **ROC Curves** - `08_roc_curves.png` (276.11 KB)

**무엇을 보여주나?**
- 3개 모델의 분류 성능을 곡선으로 비교
- 곡선이 좌상단에 가까울수록 우수한 모델

**곡선 해석**
```
True Positive Rate (TPR)
      │     XGBoost (0.8890)
      │    ╱─────────────
      │   ╱
      │  ╱ Random Forest (0.8844)
      │ ╱ Logistic Reg (0.8511)
      │╱
      └─────────────────────→ False Positive Rate (FPR)
        Random Classifier (대각선)
```

**AUC 값의 의미**
- **XGBoost: 0.8890** ⭐ (최고 - 거의 완벽)
- **Random Forest: 0.8844** ✓ (매우 우수)
- **Logistic Regression: 0.8511** ✓ (우수)

**발표 시 강조**
"XGBoost의 AUC가 0.8890으로, 임의의 표본을 선택했을 때 대출자를 비대출자보다 정확히 분류할 확률이 88.9%입니다."

### 3️⃣ **Precision-Recall Curve** - `09_precision_recall_curve.png` (160.89 KB)

**무엇을 보여주나?**
- Precision과 Recall의 트레이드오프 시각화
- 우상단에 가까울수록 우수한 모델

**곡선 해석**
```
Precision (정밀도)
      │
      │    XGBoost
      │   ╱────────╲
      │  ╱          ╲
      │ ╱ Random     ╲
      │╱────Forest───╲
      │╱    LogReg    ╲
      └─────────────────→ Recall (재현율)
```

**실용적 의미**
- **높은 Recall** (우상향): 거의 모든 대출자를 식별 ✓
- **높은 Precision** (좌상향): 대출자 예측의 정확성 높음 ✓
- **XGBoost의 특징**: Recall에 최적화 (리스크 놓치지 않음)

**발표 시 강조**
"금융기관의 관점에서 신용 리스크를 놓치는 것이 승인해서 손해 보는 것보다 중요합니다. XGBoost는 97.89%의 높은 Recall로 이를 달성합니다."

### 6️⃣ **Correlation Heatmap** - `12_correlation_heatmap.png` (184.19 KB)

**무엇을 보여주나?**

- 파란색: 음의 상관관계
- 빨간색: 양의 상관관계

**상관계수 해석**
```
상관계수 범위:
  -1 ← 음의 상관 ← 0 → 양의 상관 → +1

우리 데이터:
최대 절대값: 0.361 (ratio ↔ loan) - 중간 정도

특성 간 다중공선성:
최대: -0.259 (prop ↔ age) - 낮음 (문제 없음) ✓
```

**주요 발견**
- ✅ 특성 간 높은 상관관계 없음 (다중공선성 문제 없음)
- ✅ 모든 특성이 독립적 (중복 정보 없음)
- ✅ 각 특성이 다른 정보 제공

**발표 시 강조**
"특성 간 상관계수가 0.26 이하로, 다중공선성 문제가 없어 안정적인 모델입니다."

>
> XGBoost의 AUC가 0.8890으로 가장 높은데, 이는 무작위로 선택한 대출자와 비대출자 중에서 대출자를 더 정확히 식별할 확률이 88.9%라는 의미입니다."

>
> - 16,600명의 실제 대출자 중 16,600명을 올바르게 식별했습니다 (97.89% 재현율)
